# Fraud Detection with Machine Learning

This project focuses on detecting fraudulent e-commerce and banking transactions using machine learning techniques.

## Task 1: Data Analysis & Preprocessing
- Data cleaning and validation
- Exploratory Data Analysis (EDA)
- IP address to country mapping (geolocation analysis)
- Feature engineering (time-based & behavioral features)
- Analysis of class imbalance and resampling strategy

Datasets are excluded from version control for privacy and size reasons.
